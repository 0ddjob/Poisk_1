## [Monochrome Video to RCA Converter](/Poisk_1_Video_DIN-to-RCA)
A simple converter board convert the physical interface from a 5-pin DIN to standard RCA sockets.<br>

- Pin 1 = audio
- Pin 2 = GND
- Pin 3 = +12V
- Pin 4 = +5V
- Pin 5 = video (monochrome)
