# Poisk 1
Information about the Ukrainian SSR IBM PC/XT compatible.

![Poisk 1 case](/Poisk_1_Case.jpg)

## [Analogue RGB to VGA Converter](/Poisk_1_RGB_DIN-to-VGA)
A simple converter board convert the physical interface from a 7-pin DIN to standard VGA & 3.5mm sockets.<br>

- Pin 1 = red
- Pin 2 = GND
- Pin 3 = green
- Pin 4 = blue
- Pin 5 = composite sync.
- Pin 6 = audio
- Pin 7 = intensity

![3D view of converter board](/Poisk_1_RGB_DIN-to-VGA/Poisk_1_RGB_DIN-to-VGA_3D.png)

## [Monochrome Video to RCA Converter](/Poisk_1_Video_DIN-to-RCA)
A simple converter board convert the physical interface from a 5-pin DIN to standard RCA sockets.<br>

- Pin 1 = audio
- Pin 2 = GND
- Pin 3 = +12V
- Pin 4 = +5V
- Pin 5 = video (monochrome)

![3D view of converter board](/Poisk_1_Video_DIN-to-RCA/Poisk_1_Video_DIN-to-RCA_3D.png)
